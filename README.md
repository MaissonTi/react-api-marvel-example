# react-api-marvel-example

Simple application example using api marvel

# Pre-requisites
- Create login em marvel developer (https://developer.marvel.com/)

# Usage
- To begin with, the following steps should be followed:
- Clone repository
- Run `yarn install`
- Run `yarn run start`
